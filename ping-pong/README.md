# Ping-Pong Example   
Send data from primary to secondary.   
In the secondary, the characters are converted and sent back.   

- ESP32 is Primary
```
+-----------+            +-----------+           +-----------+
|           |            |           |           |           |
| Secondary |<==(LoRa)===|  SX126x   |<--(SPI)---|   ESP32   |
|           |            |           |           |  Primary  |
|           |            |           |           |           |
|           |===(LoRa)==>|           |---(SPI)-->|           |
|           |            |           |           |           |
+-----------+            +-----------+           +-----------+
```

- ESP32 is Secondary 
```
+-----------+             +-----------+           +-----------+
|           |            |           |           |           |
|  Primary  |===(LoRa)==>|  SX126x   |---(SPI)-->|   ESP32   |
|           |            |           |           | Secondary |
|           |            |           |           |           |
|           |<==(LoRa)===|           |<--(SPI)---|           |
|           |            |           |           |           |
+-----------+            +-----------+           +-----------+
```
